# AI_Template

This project aims to demontrate Agent Orchestration with tool calling, handoffs after completion, hook interception with logs and custom mcp tool creation. Everything using only .NET 10!!

Quick Start:

1. Upon Start you should see 2 new agents in Visual Studio Code: Pirate Captain and Brown Nose Pirate and a new mcp tool treasure-finder

2. Ask the Pirate Captain to find you a treasure and you will see the whole crew orchestration 

3. Inspect the log console_log.txt to see all hooks in action with results and check that mcp tool find-treasure was called
